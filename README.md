# Eye Health Referral Management System

A web-based referral management system for eye health services.

## Tech Stack
- Frontend: Vue.js + Vite
- Backend: Node.js 
- Database: PostgreSQL
- Docker: docker-compose

## Project Structure
- `frontend/` - Vue client app
- `backend/` - Node.js server/API
- `public/` / `src/` - shared or root frontend structure (if used)
- `docker-compose.yml` - container setup (optional)

## Requirements
- Node.js (LTS recommended)
- npm (or yarn/pnpm)
- Docker + Docker Compose
